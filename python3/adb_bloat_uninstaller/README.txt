WARNING!!!

If you are going to use the package list `vivo.json`, make sure that you have
alternative software installed first, since IT WILL DISABLE DEFAULT SYSTEM PACKAGES!

Example Essentials:
- Default Browser (com.vivo.browser/com.android.chrome)
- Default File Manager (com.android.filemanager)
- Default Keyboard (com.emoji.keyboard.touchpal.vivo)
- Default Launcher (com.bbk.launcher2)

Example Non-Essentials:
- Calculator (com.android.bbkcalculator)
- Default Launcher Themes (com.bbk.theme)
- Google Duo (com.google.android.apps.tachyon)
- Google News (com.google.android.apps.magazines)
- Google Play Music (com.google.android.music)
- Google TV (com.google.android.videos)
- Music Player (com.android.bbkmusic)
- Notes (com.android.notes)
- Video Player (com.android.VideoPlayer)
